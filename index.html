About

DAWoC is a month long programme exculsively for students of DA-IICT who are new to open source software development.to get started with open source contributions. The program encourages students to get engaged with open source and also additionally prepares them for a lot of open source programs, like Google Summer of Code aka GSoC and many more.

Open Source FTW!
